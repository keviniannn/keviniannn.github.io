FLETCHICS

ABOUT

Fletchics is a web extension that allows for users to input text and save them as notes within the extension, creating a list. After text is added, users are able to search within the list for specific letters. Matching cases will shorten the list down and highlight those matching letters, making it easy for users to narrow down the list.

This program was created using HTML, CSS, and JavaScript.
